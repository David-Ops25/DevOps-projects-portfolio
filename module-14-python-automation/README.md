# Module 14 — Python Automation on AWS

This module mirrors the structure and professionalism of **Module 13**.
It focuses on **operational automation**, **incident recovery**, and **AWS cost hygiene** using Python.

## Objectives
- Automate AWS operational tasks with Python
- Detect and recover from real failures
- Practice safe cleanup of cloud resources

## Structure
- `projects/` — individual automation demos
- `docs/` — commands, challenges, and solutions
- `tests/` — basic validation tests

Each project is self-contained with its own README.
