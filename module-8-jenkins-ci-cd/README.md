# Module 8 – Jenkins CI/CD with Docker & Nexus

This module demonstrates a complete Jenkins CI pipeline.

## What this module covers
- Jenkins running in Docker
- Pipeline as Code (Jenkinsfile)
- Docker image build automation
- Secure push to Nexus Docker registry

## CI Flow
GitHub → Jenkins → Docker Build → Nexus Registry

## Outcome
Production-ready CI pipeline aligned with real-world DevOps practices.
